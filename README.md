# 🍲 Odin Recipes

This is my first HTML project from [The Odin Project](https://www.theodinproject.com/).  
It is a simple recipe website built using only HTML.

---

## 📌 Project Description

The goal of this project was to practice basic HTML skills such as:

- Creating HTML boilerplate structure
- Using headings, paragraphs, and lists
- Adding links between pages
- Structuring a multi-page website
- Working with images

---

## 🍽️ Features

- Home page with links to recipes
- Individual recipe pages
- Simple clean HTML structure
- Beginner-friendly layout

---

## 🛠️ Built With

- HTML5

---

## 📚 What I Learned

Through this project I learned:

- How websites are structured using HTML
- How to link multiple pages together
- How to organize files in a project folder
- How to use GitHub to store and upload projects
